# Architecture Description

The full framework contains three independent feature extraction branches:

| Branch | Input | Backbone |
|--------|--------|--------|
| PSSM branch | Evolutionary profile | BiLSTM |
| CMT branch | Contact matrix | CNN / Transformer |
| Sequence branch | Protein sequence | ESM2 + BiLSTM |

The logits generated by each branch are concatenated and passed to a fusion classifier.

## Distillation Strategy

The teacher model corresponds to the full fusion framework, while the student model receives only the raw sequence.

$$
L = \alpha L_{hard} + (1-\alpha)L_{soft}
$$
